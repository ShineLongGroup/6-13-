# ShineLong Next.js Homepage

This is a Next.js-format homepage reference for ShineLong's B2B foreign trade website.

## Pages included

- `app/layout.tsx`: global metadata, Open Graph and root layout.
- `app/page.tsx`: homepage reference with Hero, trust data, products, applications, factory strength, SEO/AIO/GEO content, downloads and inquiry form.
- `app/globals.css`: responsive styling using ShineLong brand colors.
- `public/assets`: logo, factory image, founder image and catalog PDF.

## Run locally

Install dependencies first, then run:

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.
